Contoh Testing

Method: GET
URL: localhost:3000/users/
*Perlu Bearer Token
**bentuk response ada user_email, user_name, dan link user_image

Method: POST
URL: localhost:3000/users/signup
Body: (JSON)
{
  "email": "test@example.com",
  "username": "testuser",
  "password": "testpassword"
}
**response berupa status

Method: POST
URL: localhost:3000/users/signin
Body: (JSON)
{
  "email": "test@example.com",
  "password": "testpassword"
}
**response berupa status dan token JWT

Method: POST -> Dibuka di browser
URL: localhost:3000/users/google
**response berupa status dan token JWT

Untuk user sudah selesai, tinggal setting untuk pengaturan in-app dsb.
